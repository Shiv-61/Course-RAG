## Grading Rubric

### Eligibility Correctness (Prerequisite Checks)
- **Correct:** The agent references actual module/lesson ordering from the syllabus and correctly determines whether the prerequisite is met or not.
- **Partially Correct:** The agent gives the right answer but relies on general knowledge instead of catalog citations.
- **Incorrect:** The agent gives the wrong eligibility determination.

### Citation Quality
- **Full Citation:** Response includes specific PDF filename(s) and page number(s).
- **Partial Citation:** Response references documents but lacks page numbers.
- **No Citation:** Response provides no document references (acceptable ONLY for trick questions).

### Abstention Accuracy
- **Correct Abstention:** Agent clearly refuses to answer and explains that the information is not in the catalog, plus provides guidance on where to find the answer.
- **Incorrect Abstention:** Agent either fabricates an answer for a trick question, or abstains when the answer IS in the catalog.

## Analysis and Observations

### Strengths
1. **High Citation Coverage (73.3%):** 22 out of 30 responses include specific document citations with PDF names and page numbers. All queries where catalog data exists received citations.
2. **Perfect Abstention Accuracy (100%):** All 6 trick questions were correctly identified as out-of-scope, with clear explanations and actionable alternative guidance provided.
3. **Multi-hop Reasoning:** The prerequisite chain questions demonstrated the system's ability to trace dependencies across multiple lessons and modules, citing 2-6 source documents per response.
4. **Structured Responses:** All responses follow a consistent format (Answer → Why → Citations → Clarifications → Assumptions) making them easy to evaluate and audit.

### Areas for Improvement
1. **Response Time:** Average response time of 95.8s could be improved by optimizing the agent pipeline or using a faster model tier.
2. **Cross-Module Prerequisites:** The catalog lacks explicit cross-module prerequisite declarations, requiring the system to infer dependencies from lesson ordering.
3. **Student Profile Integration:** Some planning queries could benefit from student profile data (completed courses, grades) for more personalized recommendations.

### Conclusion
The Course-RAG system demonstrates strong performance across all four evaluation categories, achieving a **73.3% citation coverage rate** and **100% abstention accuracy**. The multi-agent pipeline (Intake → Retriever → Planner → Verifier) effectively grounds responses in catalog data while safely abstaining from out-of-scope questions. The system is well-suited for academic advising use cases where grounded, cited responses are critical.
